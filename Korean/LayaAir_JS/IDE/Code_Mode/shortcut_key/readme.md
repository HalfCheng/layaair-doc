###단축키 사용자 정의

레이어이드의 코드 부분 단축키는 vscode 기능을 계승했다.개별 단축키가 다르다.사용자는 자신의 취향에 따라 설정할 수 있습니다. 개발자가 단축키를 변경하려면 다음 단계에 따라 동작하십시오:



**메뉴-파일-첫번째 옵션-키보드 단축식.**
