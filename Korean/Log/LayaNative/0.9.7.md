### 新特性

1, 자원을 zip 으로 포장하고, 로컬 으로 다운로드 해제, 이 기능을 통해 자원을 한 번에 업데이트할 수 있습니다.자세한 문서: https://github.com/layabox/layair-doc/tree/master/Chinese/LayaNative/resourceupdate

2, 지원ies 시스템이 기본 글꼴에 포함되어 있으며, 동작과 android.자세한 문서: https://github.com/layabox/layair-doc/tree/master/Chinese/LayaNative/built

3. xmlHtttpRequest 동적 글꼴을 사용하여 글꼴을 설정할 수 있습니다.자세한 문서: https://github.com/layabox/layair-doc/tree/master/Chinese/LayaNative/extent-font

4, 많은 apk 가방을 지원해 GooglePlay 패키지 크기가 100MB를 넘어 확장패를 요구하는 문제를 해결했다.자세한 문서: https://github.com/layabox/layair-doc/tree/master/Chinese/LayaNative/apk u expansion

5. Android가 자동으로 apk 을 업데이트한 코드.자세한 문서: https://github.com/layabox/layair-doc/tree/master/Chinese/LayaNative/apk-update

###복구 버그

1. sprite.skew bug 복구.

2. http 머리는 항목이 너무 많아 crash 의 bug 을 복구했다.

3. loadingView 때문에 일부 휴대전화 화면이 문제를 맞추는 버그 를 해결했다.

4. img 이 쓰레기로 회수돼 onload 사건에 호출되지 않은 bug 을 수정했다.

5, ide 이선 복구, 프로젝트를 구축할 수 없는 bug.

6, ios 는 극단적인 상황에서 화면을 클릭해 터치의 호응이 없는 bug 을 복구했다.
