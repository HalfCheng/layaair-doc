#AS 3プロジェクト分包説明

>author：charley

###1、なぜカバンを分けますか？

Layabox独自のコンパイラはAS 3言語で書かれたプロジェクトをjsコードにコンパイルすることができます。しかし、デフォルトでは、すべてのコードがjsにコンパイルされます。プロジェクトが大きいなら、コンパイル後のjsも大きくなります。jsが負荷体験に影響を与えるほど大きいなら、パケットを分けて、いくつかの独立した後期機能jsを必要に応じてロードします。前の段階の負荷圧力を減らすことができます。
