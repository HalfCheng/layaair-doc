###프로젝트 관리자

프로젝트 관리자 창에 나무 모양으로 항목 폴더의 모든 코드 및 자원 구조를 보여 줍니다.폴더가 자동으로 조작 시스템에서 폴더 내용에 대한 수정을 동기화합니다.신축, 개명, 삭제 등 상용 조작 기능도 가능하다.
